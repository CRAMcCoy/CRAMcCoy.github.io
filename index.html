The portfolio will begin with an introduction of the artefact requirements and challenges. Then it will describe the reverse engineering, CAD, and fabrication (as shown in figure 1) with an appraisal of techniques explored and used at each stage. Finally, a reflective evaluation in which will summarise the development and key points of the portfolio will be delivered.

Figure 1: Project plan

The content will be arranged across several webpages to allow for images and STL files to be displayed. At the top of each page there will be the title of the current webpage and a link to the main page, then at the bottom of the page there will be a link to the next webpage to be read.


Note to the reader:

The portfolio will refer to the project as “the hand”, which is not to be confused with the hand being replicated (“the original hand”) or the combination of this product with the exoskeleton and the skin (“the finished product”). Where the author deviates from this rule, the context will make this unambiguous.
